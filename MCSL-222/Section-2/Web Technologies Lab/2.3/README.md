# 2.3 PROJECT CREATION USING IDE

In this section, we will try to create multiple projects either in Apache NetBeans or 
Eclipse. Before you start with the practical sessions, let us see how to create a java 
project in Apache NetBeans and Eclipse first. 
Create Project in Apache NetBeans: 
     
**Step 1**: Open the Apache NetBeans 12.4 IDE. 
**Step 2**: New Project: Click on File Menu  New Project (Wizard shown in figure - 
20 will display) 
**Step 3**: Finding Features(One time job) 
This option asks us to activate the J2EE features in IDE once after fresh installing of 
Apache Netbean IDE as shown in figure – 2.20. Click on download and 
activate/enable the required plugin/features for Java Web Application development.
Once the process is finished, the Wizard for New Web application will be displayed as shown in figure – 2.21. Skip step 3 if already Plugin activated.
**Step 4**: Click on Java with Ant  Java Web in Categories and select Web 
Application in Projects section then click on Next button as shown in figure -2.21.
**Step 5**: Name and Location or Project 
In this wizard we have to enter Project Name, Project location where project source code to be stored/located, Library folder; if external dedicated libraries location needs to be added in the Project. In this case Only Project name is changed from the default value, the rest fields are as default, as shown in figire- 2.22 . Then click on Next to procced.
